# Quy ước Status

## 1. USERS
| Giá trị | Ý nghĩa |
|--------|---------|
| `ACTIVE` | Tài khoản hoạt động |
| `INACTIVE` | Tài khoản bị vô hiệu hoá |
| `LOCKED` | Khoá tạm thời |

## 2. PURCHASE_ORDERS
| Giá trị | Ý nghĩa |
|--------|---------|
| `NEW` | Đơn mới tạo |
| `PENDING_APPROVAL` | Chờ duyệt |
| `APPROVED` | Đã duyệt |
| `DELIVERED` | Đã giao hàng |
| `CLOSED` | Đã hoàn tất |
| `CANCELLED` | Huỷ |
